Getting Started with NLTK

Programming with NLTK

The NLTK libraries consists of a set of Python modules, each of which defines classes and functions related to a single data structure or task. Before you can use a module, you must import its contents. The simplest way to import the contents of a module is to use the "from module import *" command. For example, to import the contents of the tagging module, type:

>>> from nltk_lite.tag import *

For more information on importing modules, and for a general introduction to NLTK-Lite, see the tutorials.

Demonstrations

NLTK includes three graphical parser demonstrations:

  • nltk_lite.draw.rdparser: A top-down recursive-descent parser demo.
  • nltk_lite.draw.srparser: A bottom-up shift-reduce parser demo.
  • nltk_lite.draw.chart: A chart-parser demo.

These can be run from Python by importing them and calling the demo() function: 

>>> from nltk_lite.draw.rdparser import *
>>> demo()

Alternatively, the demonstrations can be run from Windows by double-clicking on their Python source files; or from UNIX or OS X by running Python on their source files: 

$ python <path-to-nltk>/nltk_lite/draw/rdparser.py

Running Python

There are three typical ways to use NLTK with Python:

  • Run the Python interpreter, and use NLTK interactively: 

    $ python
Python 2.2.2 (#1, 02/19/03, 17:56:57) 
[GCC Apple cpp-precomp 6.14] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> from nltk_lite.corpora import brown
>>> text = brown.raw('a')
...

  • Use a text editor to create a program; and execute it (either from the command line, or by double-clicking on its icon). 

    $ python my_program.py
...

  • Use an integrated development enviroment, such as IDLE or Emacs' Python Mode.

How you prefer to interact with Python and NLTK is a personal choice. If you are new to Python, we suggest that you try using an integrated development enviroment, such as IDLE. IDLE is included with Python on most systems.

Running IDLE

On Windows, IDLE is located in the start menu ("Programs→Python 2.4→Idle"). On unix, you can run IDLE from the command line, with the command "idle".

Within IDLE, you can use the interactive interpreter; or you can create a new Python file ("File→New Window"). To execute the new Python file, use "Edit→Run script" (or Ctrl-F5). To save the new Python file, use "File→Save" (or Ctrl-S). When you save your Python file, use the extension ".py". You can open an existing Python program with "File→Open" (or Ctrl-O).

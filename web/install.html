Installing NLTK-Lite

NLTK-Lite requires Python 2.4 or higher, the Numpy math library, and packages downloaded from this site. WordNet 2.1 may also be installed. For detailed, platform-specific instructions, please see one of the following guides:

  • Binary installation on Mac
  • Binary installation on Windows
  • Source installation on Unix and Mac
  • Binary RPM installation on Linux:
         Python, Numarray (src only), NLTK-Lite (rpm, data, doc)
  • Debian Packages for NLTK-Lite

NLTK-Lite CD-ROM

An ISO image is available on the download page; it contains Python, NLTK-Lite, and all the other required packages for all platforms.

Latest Development Version

To access the latest development version of NLTK-Lite, check out the repository, build and install the system python setup.py install, and set the NLTK_LITE_CORPORA environment variable to point to the top level corpora directory.

Installing the original NLTK

The original NLTK is no longer supported. If you want it anyway, download it here.

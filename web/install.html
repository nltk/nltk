Installing NLTK

NLTK requires Python 2.4 or higher, and packages downloaded from this site. Detailed instructions are available on the NLTK wiki and in the top level of the CD-ROM.

NLTK CD-ROM

An ISO image is available on the download page; it contains Python, NLTK, and all the other required packages for all platforms.

Latest Development Version

To access the latest development version of NLTK, check out the repository, build and install the system sudo python setup.py install, and set the NLTK_LITE_CORPORA environment variable to point to the top level corpora directory.

Installing the original NLTK

The original NLTK is no longer supported. If you want it anyway, download it here.

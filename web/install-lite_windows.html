Installing NLTK-Lite on Windows

1. Installing Python

NLTK-Lite requires the Python scripting language interpreter to be installed on your machine.

Download the Python source distribution from http://www.python.org/ftp/python/2.5/python-2.5.msi?download

Launch the installer by double clicking on the file. Follow the installation instructions.

2. Installing Numpy

NLTK also requires a Python module called Numpy, which provides a range of mathematical functions.

Download Numpy from http://prdownloads.sourceforge.net/numpy/numpy-1.0.win32-py2.5.exe?download

Launch the installer by double clicking on the file. Follow the installation instructions.

3. Installing NLTK-Lite

Download the NLTK-Lite source distribution from http://prdownloads.sourceforge.net/nltk/nltk_lite-0.7b1.win32.exe?download

The installation "wizard" will guide you through the installation process. If you are upgrading to a new version of NLTK-Lite, the installation wizard will ask you if you wish to overwrite the old NLTK-Lite files; answer "yes."

4. Installing the NLTK-Lite Corpora Package

Download the NLTK-Lite Corpora distribution from http://prdownloads.sourceforge.net/nltk/nltk_lite-corpora-0.7b1.zip?download

Uncompress the NLTK-Lite corpus package. Typically, this is done by double-clicking on the nltk_lite-corpora-??.??.zip file. However, depending on what tool you use to unzip files, the procedure might be slightly different. If you don't already have a tool to unzip files, you can use the WinZip evaluation version.

Extract the corpora to a folder on your hard drive. Common locations for this folder are C:\Program Files\Python2.5\nltk-lite; and C:\Program Files\Python2.5\lib\nltk-lite.

If the NLTK-Lite corpora folder that you created is not in path-to-python\nltk-lite or path-to-python\lib\nltk-lite, then you will need to tell NLTK-Lite where to find the corpora. You can do this by setting the NLTK_LITE_CORPORA environment variable. For most versions of Windows, this enviroment variable can be set with the following procedure:

  1. Right click on "My Computer", and select "Properties".
  2. Select the "Advanced" tab.
  3. Click on "Environment Variables"
  4. Under "User Variables," click "New..."
  5. Type "NLTK_LITE_CORPORA" into the "Variable" field; and type the path to the NLTK corpora folder that you created into the "Value" field.
  6. Hit "OK" (2 times) to close the dialog windows.

5. Installing WordNet (optional)

Download the WordNet binary distribution from http://wordnet.princeton.edu/2.0/WordNet-2.0.exe. The installation "wizard" will guide you through the installation process.

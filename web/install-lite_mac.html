Installing NLTK-Lite on Macintosh (OS X)

1. Installing Python

NLTK-Lite requires the Python scripting language interpreter to be installed on your machine.

Download the Python binary from http://www.python.org/ftp/python/2.5/python-2.5-macosx.dmg to /tmp.

Mount the dmg, double click and follow the OSX Packager install instructions.

You may now have 2 versions of Python installed on your Mac:

  • Python 2.3 or 2.4, installed with Mac OS X by default in 
    /usr/bin/python
  • Python 2.5, installed by the previous step in 
    /usr/local/bin/python
This is significant since NLTK requires Python 2.4 or later to run.

Check the Python installation: 

	python -V
which should return something similar to: 
Python 2.5
.

(You may need to add a symbolic link in order to get the correct version of Python.)

2. Installing Numpy

NLTK also requires a Python module called Numpy, which provides a range of mathematical functions. This must be installed from source.

Download the Numpy source distribution from http://prdownloads.sourceforge.net/numpy/numpy-1.0.tar.gz?download to /tmp. NUMPY = $(SFNETMIRROR)/numpy wget -N -P python/unix/ $(NUMPY)/

Uncompress the Numpy source distribution: 

	cd /tmp
	gunzip numpy-1.0.tar.gz
	tar -xvf numpy-1.0.tar

Install the Numpy module: 

	cd numpy-1.0
	sudo python setup.py install

3. Installing NLTK-Lite

Download the NLTK-Lite source distribution from http://prdownloads.sourceforge.net/nltk/nltk_lite-0.7a2.tar.gz?download to /tmp.

Uncompress the NLTK-Lite source distribution: 

	cd /tmp
	gunzip nltk_lite-0.7a2.tar.gz
	tar -xvf nltk_lite-0.7a2.tar

Install the NLTK-Lite module: 

	cd nltk_lite-0.7a2
	sudo python setup.py install

4. Installing the NLTK-Lite Corpora Package

Download the NLTK-Lite Corpora distribution from http://prdownloads.sourceforge.net/nltk/nltk_lite-corpora-0.7a2.zip?download to /tmp.

Uncompress the NLTK-Lite Corpora distribution: 

	cd /tmp
	unzip nltk_lite-corpora-0.7a2.zip

Move the corpora to a standard directory, e.g. /usr/share/nltk_lite, or /usr/local/share/nltk_lite. If you are not a system administrator, you can put this directory in your home directory (e.g. ~/nltk_lite). 

	mv /tmp/nltk_lite-corpora /usr/share/nltk_lite

Set an environment variable for the NLTK-Lite Corpora 

	export NLTK_LITE_CORPORA=/usr/share/nltk_lite

For convenience this environment variable can be set in your $HOME/.bash_profile so it is loaded every time you start a terminal session: 

	echo "export NLTK_LITE_CORPORA=/usr/share/nltk_lite" >> $HOME/.bash_profile

Alternatively, you can also use the function set_basedir() in the module nltk_lite.corpora: 

	[user@host ~]$ python
	Python 2.4.1 (#1, Aug  4 2005, 10:25:27)
	[GCC 3.4.3 20050227 (Red Hat 3.4.3-22.fc3)] on linux2
	Type "help", "copyright", "credits" or "license" for more information.
	>>> from nltk_lite.corpora import set_basedir
	>>> set_basedir('/some/other/path')

5. Installing WordNet (optional)

Download the WordNet source distribution from http://wordnet.princeton.edu/2.0/WordNet-2.0.tar.gz to /tmp.

Open a terminal session, and uncompress the distribution: 

	cd /tmp
	gunzip WordNet-2.0.tar.gz
	tar -xvf WordNet-2.0.tar

Compile and install the Wordnet source distribution: 

	cd Wordnet-2.0
	./configure
	make
	sudo make install

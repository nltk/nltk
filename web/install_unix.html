Installing NLTK on Unix or Macintosh (OS X)

Installing NLTK

Download the NLTK source distribution (nltk-??.??.tar.gz), and use setup.py to install it: 

[/tmp]$ gunzip nltk-??.??.tar.gz 
[/tmp]$ tar -xf nltk-??.??.tar
[/tmp]$ cd nltk-??.??
[/tmp/nltk-??.??]$ su
Password:
[/tmp/nltk-??.??]# python setup.py install

Installing the NLTK Data Package

  1. Download the NLTK data package (nltk-data-??.??.zip).
  2. Uncompress the NLTK data package. 
    3. [/tmp]# unzip nltk-data-??.??.zip 
[/tmp]# cd nltk-data-??.??
[/tmp/nltk-data-??.??]#
  3. Create a directory to hold the NLTK corpus set. A common path for this directory is /usr/share/nltk. If you are not a system administrator, you can put this directory in your home (e.g. ~/nltk). 
    [/tmp/nltk-data-??.??]# mkdir /usr/share/nltk
  4. Uncompress each corpus that you'd like to use to into the NLTK corpus directory that you created. 
    [/tmp/nltk-data-??.??]# unzip <corpus>.zip -d /usr/share/nltk
  5. If the NLTK corpus directory that you created is not in /usr/lib/nltk, /usr/local/lib/nltk, or /usr/share/nltk, then you will need to tell NLTK where to find the data. You can do this by setting the NLTK_CORPORA environment variable: 
    For sh or bash:
  [/tmp]# export NLTK_CORPORA=/some/where/nltk

For csh or tcsh:
  [/tmp]# setenv NLTK_CORPORA /some/where/nltk
    To make this change persistent, you will need to edit your shell's startup file: 
    For sh or bash:
  [/tmp]# echo 'export NLTK_CORPORA=/some/where/nltk' >>~/.profile

For csh or tcsh:
  [/tmp]# echo 'setenv NLTK_CORPORA /some/where/nltk' >>~/.cshrc
    Alternatively, you can also use the function set_basedir() in the module nltk.corpus: 
    >>> from nltk.corpus import set_basedir
>>> set_basedir('/some/where/nltk')

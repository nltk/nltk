Installing NLTK on RedHat Linux

Installing NLTK

If you are running RedHat 7.1 (or a compatible system), you can install NLTK from the RPM file:

[/tmp]$ su
Password:
[/tmp]# rpm -i nltk-??.??-1.noarch.rpm

If you wish to build an RPM for NLTK on a system other than RedHat 7.1, you can use the SRPM file:

[/tmp]$ rpm --rebuild nltk-??.??-1.noarch.rpm

Installing the NLTK Data Package

The NLTK data package is not available as an RPM; see the UNIX instructions for installing the data package.

Installing the NLTK Contributions Package

Download the RPM file for the NLTK contributions package, and use rpm to install it:

[/tmp]$ su
Password:
[/tmp]# rpm -i nltk-??.??-1.noarch.rpm

If you wish to build an RPM for the contributions on a system other than RedHat 7.1, you can use the SRPM file:

[/tmp]$ rpm --rebuild nltk-??.??-1.noarch.rpm

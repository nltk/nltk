NLP Toolkit Timetable

This page indicates the current and projected status of the toolkit. For this purpose, the toolkit is broken up into units, each of which includes contains several Python modules and tutorials. The core unit provides basic data types and processing systems that are used througout the toolkit. Each of the other units is focused on a specific task or domain:

  • Tagging: Augmenting each token of a text with supplementary information, such as its part of speech or its WordNet synset tag.
  • Parsing: Producing trees that represent the structures of texts.
  • Text Classification: Assigning categories to texts, based on their content.
  • Machine Translation: Mapping a text to a semantically equivalant text in a different language.
  • Information Retrieval: Determining which documents are most relevant for a given query.
  • Language Modelling: Predicting the likelihood of a text.
  • Speech Processing: Speech recognition and generation, signal processing, filtering, etc.

Installing NLTK-Lite for Linux

1. Installing Python

NLTK-Lite requires the Python scripting language interpreter to be installed on your machine.

Installing Python from RPMs

Download and install the basic Python package with rpm:

        rpm -Uvh http://download.fedora.redhat.com/pub/fedora/linux/core/4/i386/os/Fedora/RPMS/python-2.4.1-2.i386.rpm

Alternatively, you can use a package manager such as yum to do the same:

        yum install python

Installing Python from Source

Download the Python source distribution from http://www.python.org/ftp/python/2.5/Python-2.5.tgz to /tmp.

Open a terminal session, and uncompress the Python source distribution: 

	cd /tmp
	gunzip Python-2.5.tar.gz
	tar -xvf Python-2.5.tar

Compile and install the Python source distribution: 

	cd Python-2.5
	./configure
	make
	sudo make install

Check the Python installation: 

	python -V

2. Installing Numarray

NLTK also requires a Python module called Numarray, which provides a range of mathematical functions.

Installing Numarray from RPMs

There is no official RPM for Numarray. It can be installed from source.

Installing Numarray from Source

Download the Numarray source distribution from http://prdownloads.sourceforge.net/numpy/numarray-1.5.2.tar.gz?download to /tmp.

Uncompress the Numarray source distribution: 

	cd /tmp
	gunzip numarray-1.5.2.tar.gz
	tar -xvf numarray-1.5.2.tar.gz

Install the Numarray module: 

	cd numarray-1.5.2
	sudo python setup.py install

3. Installing NLTK-Lite

Installing NLTK-Lite from RPMs

Download the NLTK-Lite RPM from http://prdownloads.sourceforge.net/nltk/nltk_lite-0.6.6-1.noarch.rpm?download to /tmp.

You can now install the RPM as follows:

        rpm -Uv /tmp/nltk_lite-0.6.6-1.noarch.rpm

Installing NLTK-Lite from Source

Download the NLTK-Lite source distribution from http://prdownloads.sourceforge.net/nltk/nltk_lite-0.6.6.tar.gz?download to /tmp.

Uncompress the NLTK-Lite source distribution: 

	cd /tmp
	gunzip nltk_lite-0.6.6.tar.gz
	tar -xvf nltk_lite-0.6.6.tar

Install the NLTK-Lite module: 

	cd nltk_lite-0.6.6
	sudo python setup.py install

4. Installing the NLTK-Lite Corpora Package

The NLTK-Lite package nltk_lite.corpora is meant to be used with various corpus materials that are bundled separately from NLTK-Lite.

Installing NLTK-Lite Corpora from RPMs

It is currently only possible to install the NLTK-Lite Corpora Package from source.

Installing NLTK-Lite Corpora from Source

Download the NLTK-Lite Corpora distribution from http://prdownloads.sourceforge.net/nltk/nltk_lite-corpora-0.6.6.zip?download to /tmp.

Uncompress the NLTK-Lite Corpora distribution: 

	cd /tmp
	unzip nltk_lite-corpora-0.6.6.zip

Move the corpora to a standard directory, e.g. /usr/share/nltk_lite or /usr/local/share/nltk_lite. If you are not a system administrator, you can put this directory in your home directory (e.g. ~/nltk_lite). 

	mv /tmp/nltk_lite-corpora /usr/share/nltk_lite

Set an environment variable for the NLTK-Lite Corpora 

	export NLTK_LITE_CORPORA=/usr/share/nltk_lite

For convenience this environment variable can be set in your $HOME/.bash_profile so it is loaded every time you start a terminal session: 

	echo "export NLTK_LITE_CORPORA=/usr/share/nltk_lite" >> $HOME/.bash_profile

Alternatively, you can also use the function set_basedir() in the module nltk_lite.corpora: 

	[user@host ~]$ python
	Python 2.4.1 (#1, Aug  4 2005, 10:25:27)
	[GCC 3.4.3 20050227 (Red Hat 3.4.3-22.fc3)] on linux2
	Type "help", "copyright", "credits" or "license" for more information.
	>>> from nltk_lite.corpora import set_basedir
	>>> set_basedir('/some/other/path')

5. Installing WordNet (optional)

Installing WordNet from RPMs

There is no official RPM for WordNet. It can be installed from source.

Installing WordNet from Source

Download the WordNet source distribution from http://wordnet.princeton.edu/2.0/WordNet-2.0.tar.gz to /tmp.

Open a terminal session, and uncompress the distribution: 

	cd /tmp
	gunzip WordNet-2.0.tar.gz
	tar -xvf WordNet-2.0.tar

Compile and install the Wordnet source distribution: 

	cd Wordnet-2.0
	./configure
	make
	sudo make install

Next, download and install the pywordnet source distribution from http://prdownloads.sourceforge.net/pywordnet/pywordnet-2.0.1.tar.gz?download to /tmp.

Open a terminal session, and uncompress the distribution: 

	cd /tmp
	gunzip pywordnet-2.0.1.tar.gz
	tar -xvf pywordnet-2.0.1.tar

Now install the pywordnet source distribution: 

	cd pywordnet-2.0.1
	python ./setup.py install

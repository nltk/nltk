Installing the Original NLTK

The original NLTK is no longer supported. Most users should just install NLTK-Lite. If you want it anyway, download it here:

  • NLTK 1.4.4: rpm, src-rpm, tar.gz, exe, zip
  • NLTK-Contrib 1.4.4: rpm, src-rpm, tar.gz, exe, zip
  • NLTK-Data 0.3 zip

NLTK is divided into three packages, each of which can be downloaded from the NLTK download page. The NLTK package contains the Natural Language Toolkit. The NLTK data package contains a collection of corpora and sample data that can be easily used with NLTK. The NLTK contributions package contains third-party modules and packages based on NLTK that have been contributed to the NLTK project, but not integrated into the toolkit.

Installation Requirements

NLTK requires Python 2.3 or higher. To check what version of Python you are running, run "python -V". Alternatively, run an interactive Python installation session, and check which version it reports. You can download the latest version of Python from the Python download page.

NLTK uses several other open source Python packages. These packages are not required to use most of Python's core functionality. However, without them you won't be able to use some features of NLTK (such as graphical demos). The following packages are currently used by the natural language toolkit:

  • Tkinter ("Tk interface"): The standard Python interface to the Tk GUI toolkit. This is usually installed by default, but you may need to install it separately. [download RPM]
  • Numerical Python: A fast array manipulation package for Python. For Python version ≤ 2.3 and NLTK version ≤ 1.4.2, install Numeric. For Python version 2.4 and NLTK version ≥ 1.4.3, install Numarray.

Additional packages

NLTK users may find the following package(s) useful:

  • WordNet, a lexical database developed at Princeton University.
  • PyWordNet, a Python interface to WordNet.

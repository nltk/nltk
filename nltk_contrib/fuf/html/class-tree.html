Class Hierarchy

  • object: The most base type
    • nltk.featstruct.CustomFeatureValue: An abstract base class for base values that define a custom unification method.
      • fuf.fstypes.TypedFeatureValue: Feature value that has been defined as a FUF type
    • fuf.fstypes.FeatureTypeTable: The FeatureTypeTable maintains relationships between a given feature value and its specializations.
    • fuf.fuf.GrammarPathResolver: GrammarPathResolver analyses a given grammar and generates the list of possible paths through the grammar by unpacking all the possible alternations.
    • fuf.link.LinkResolver: Object for resolving links Note: links should not be resolved if 'alt' is present in the feature structure.
    • fuf.link.ReentranceLink: Used to store fuf's reentrance links; these are resolved after the parsing and alt structure generation.
    • fuf.sexp.SexpFileParser: Parse a file that contains multiple s-expressions
    • fuf.sexp.SexpListParser: Parse the text and return a a SexpList
    • fuf.statemachine.StateMachine: A basic but flexible state machine
      • fuf.statemachine.PushDownMachine: State machine that uses and additional stack.
    • fuf.fuf.Unifier: Class for unification of feature structures defined by FUF.
    • list: list() -> new list list(sequence) -> new list initialized from sequence's items
      • fuf.sexp.SexpList: A list, extracted from an s-expressoin string.

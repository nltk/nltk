Module Hierarchy

  • fuf: FUF is an interpreter, originally written by Michael Elhadad in Common Lisp for a functional unification based language specifically designed for the development of text generation applications.
    • fuf.fstypes: fstypes.py module contains the implementation of feature value types as defined in the FUF manual (v5.2)
    • fuf.fuf
    • fuf.fufconvert
    • fuf.linearizer: The linearizer for unified fuf feature structures
    • fuf.link: Module for resolving relative and absolute paths in FUF
    • fuf.sexp: Module for tokenizing and parsing s-expressions
    • fuf.specialfs: Handling for special feature names during parsing
    • fuf.statemachine: Basic state machine class
    • fuf.util: This module contains a couple of functions for better viewing of the FUF convesion/unification results

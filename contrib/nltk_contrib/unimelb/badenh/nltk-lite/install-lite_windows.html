Installing NLTK-Lite on Windows

Installing Python

NLTK-Lite requires the Python scripting language interpreter to be installed on your machine.

Download the Python source distribution from http://www.python.org/ftp/python/2.4.1/Python-2.4.1.msi

Launch the installer by double clicking on the file. Follow the installation instructions.

Installing Numarray

NLTK also requires a Python module called Numarray, whicb provides a range of mathematical functions.

Download the Numarray source distribution from http://prdownloads.sourceforge.net/numpy/numarray-1.3.3.exe?download

Launch the installer by double clicking on the file. Follow the installation instructions.

Installing NLTK-Lite

Download the NLTK-Lite source distribution from http://prdownloads.sourceforge.net/nltk/nltk_lite-0.2.exe?download

The installation wizard will guide you through the installation process. If you are upgrading to a new version of NLTK-Lite, the installation wizard will ask you if you wish to overwrite the old NLTK-Lite files; answer "yes."

Installing the NLTK-Lite Data Package

Download the NLTK-Lite Data distribution from http://prdownloads.sourceforge.net/nltk/nltk_lite-corpora-0.2.zip?download

Uncompress the NLTK data package. Typically, this is done by double-clicking on the nltk-data-??.??.zip file. However, depending on what tool you use to unzip files, the procedure might be slightly different. If you don't already have a tool to unzip files, you can use the WinZip evaluation version.

Create a folder to hold the NLTK-Lite data. Common locations for this folder are C:\Program Files\Python2.4\nltk-lite; and C:\Program Files\Python2.4\lib\nltk-lite.

Uncompress each corpus or collection of sample data that you'd like to use to into the NLTK-Lite data folder that you created. Typically, this is done by copying corpus.zip into the NLTK-Lite data folder that you created; and double-clicking on corpus.zip. Each corpus.zip file will uncompress into its own subfolder.

If the NLTK-Lite data folder that you created is not in path-to-python\nltk-lite or path-to-python\lib\nltk-lite, then you will need to tell NLTK-Lite where to find the data. You can do this by setting the NLTK_LITE_CORPORA environment variable. For most versions of Windows, this enviroment variable can be set with the following procedure:

  1. Right click on "My Computer", and select "Properties".
  2. Select the "Advanced" tab.
  3. Click on "Environment Variables"
  4. Under "User Variables," click "New..."
  5. Type "NLTK_CORPORA_LITE" into the "Variable Name" field; and type the path to the NLTK data folder that you created into the "Variable Value" field.
  6. Hit "Ok" (3 times) to close the dialog windows.

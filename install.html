[edit]

Windows

Required: Please download and install the following:

  • Python: http://www.python.org/ftp/python/2.5.1/python-2.5.1.msi
  • NLTK: http://prdownloads.sourceforge.net/nltk/nltk_lite-0.7.5.win32.exe
  • Corpora: http://prdownloads.sourceforge.net/nltk/nltk_lite-corpora-0.7.5.zip

Corpora: Unpack the corpora using a utility such as Winzip, and move the corpora folder to C:\corpora (or some other central location). If the corpora folder that you created is not in this default location then you will need to set the NLTK_LITE_CORPORA environment variable: right click on "My Computer", select Properties > Advanced > Environment Variables > User Variables > New..., then type NLTK_LITE_CORPORA into the Variable field, and type the path to the NLTK corpora folder that you created into the Value field.

Optional: Some users may need the following:

  • Numpy: http://prdownloads.sourceforge.net/numpy/numpy-1.0.2.win32-py2.5.exe (for HMM tagging and for clustering)
  • Matplotlib: http://prdownloads.sourceforge.net/matplotlib/matplotlib-0.90.0.win32-py2.5.exe (for some graphical visualizations)
  • WordNet: http://wordnet.princeton.edu/2.1/WordNet-2.1.exe (for NLTK versions prior to 0.7.5)
[edit]

Macintosh

Required: NLTK requires Python 2.4 or later. Open Finder > Applications and check for a version of MacPython. If necessary, install Python:

  • Python: http://www.python.org/ftp/python/2.5.1/python-2.5.1-macosx.dmg

Optional Some users may need the following:

  • Numpy: http://pythonmac.org/packages/py25-fat/dmg/numpy-1.0.2-py2.5-macosx10.4-2007-04-04.dmg (for HMM tagging and for clustering)
  • Matplotlib: http://pythonmac.org/packages/py25-fat/dmg/matplotlib-0.90.0-py2.5-macosx10.4-2007-02-20.dmg (for some graphical visualizations)

Now proceed with NLTK Source Installation (below).

[edit]

Linux/Unix

Required: NLTK requires Python 2.4 or later (check with python -V). If you need to install Python, you can do this using your favorite package manager, or find a suitable RPM, or download and build Python from source: http://www.python.org/ftp/python/2.5.1/Python-2.5.1.tgz

Optional Some users may need the following:

  • Numpy: http://prdownloads.sourceforge.net/numpy/numpy-1.0.2.tar.gz (for HMM tagging and for clustering)
  • Matplotlib: http://prdownloads.sourceforge.net/matplotlib/matplotlib-0.90.0.tar.gz

Install these by unpacking the source distribution and running sudo python setup.py install

Now proceed with NLTK Source Installation (below).

[edit]

NLTK Source Installation

These instructions are for Mac, Linux and Unix platforms.

Check your Python installation is adequate: open a terminal and type python -V to see what version you get; once you get the python prompt type import numpy to check you can load the numerical library (set your PATH and PYTHONPATH environment variables if necessary).

1. Download the NLTK source distribution: http://prdownloads.sourceforge.net/nltk/nltk_lite-0.7.5.tar.gz

Unpack the archive (e.g. with gunzip or Stuffit Expander). Open the terminal and cd into this new folder, and type sudo python setup.py install

2. Next, download the NLTK Corpora distribution: http://prdownloads.sourceforge.net/nltk/nltk_lite-corpora-0.7.5.zip

Unpack this archive to extract a folder called corpora, and move this into /usr/share/nltk_lite or ~/nltk_lite.

Set an environment variable to refer to this location, e.g. 

export NLTK_LITE_CORPORA=/usr/share/nltk_lite/corpora/

(this can be done in your .bash_profile file).

3. Installing WordNet (optional)

If you are installing a version of NLTK earlier than 0.7.5, you need to install Wordnet 2.1: http://wordnet.princeton.edu/2.1/WordNet-2.1.tar.gz

Unpack this archive and build and install WordNet: 

	cd WordNet-2.1
	./configure
	make
	sudo make install

Installation to Non-Standard Location

If you don't want to install NLTK in a central location, download and unpack the zip distributions:

  • http://prdownloads.sourceforge.net/nltk/nltk_lite-0.7.5.zip
  • http://prdownloads.sourceforge.net/nltk/nltk_lite-corpora-0.7.5.zip

Move NLTK to the desired location, then add this location to your PYTHONPATH. Similarly, move the NLTK Corpora to the desired location, and set this location as the value of NLTK_LITE_CORPORA
